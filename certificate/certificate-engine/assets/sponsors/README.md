# Sponsor Logos

Place your sponsor logo images in this folder.

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- Any format supported by @napi-rs/canvas

## Naming
Match the `logoPath` column in `data/sponsors.csv`

Example files referenced in sponsors.csv:
- techcorp.png
- innovation_labs.png
- digital_future.png
- brainpower.png
- etc.

## Recommendations
- Use transparent PNG files for best results
- Recommended size: 100-200px width
- Square or rectangular logos work best
- Keep file sizes under 500KB for performance
